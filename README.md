# pvim

My **PERSONAL** nvim config according to my needs

## usage

```bash
git pull https://github.com/FarikunAziz/pvim.git

#renaming
mv pvim nvim

#or create new one
mkdir -p ~/.config/nvim
```

## structure

```bash
cd ~/.config/nvim
```

```bash
nvim
├── README.md
├── init.lua
├── lazy-lock.json
├── lsp
└── lua
    ├── Lazy.lua
    ├── plugins
    │   ├── themes
    └── user
```
